## 📝 ประวัติการทำงาน (Changelog)

### [อัปเดตครั้งที่ 2] - 9 มิถุนายน 2026
**รายละเอียด:** ดำเนินการเพิ่มฟีเจอร์ใหม่ 9 ข้อ ตามที่ผู้ใช้งานร้องขอ
**สถานะ:** ✅ เสร็จสมบูรณ์

**รายการที่ดำเนินการ:**
1. **หน้าแรก (Home Page):**
   - เพิ่มระบบ Slider สลับระหว่าง 2 หน้าต่าง คือ "แบนเนอร์เชิญชวนบริจาค (พร้อมสถิติ)" และ "ตารางเวลาละหมาดประจำวัน"
2. **ระบบสถิติการบริจาค (Donation Statistics):**
   - สร้าง API `server/api/donations/stats.ts` เพื่อรวมยอดเงินและจำนวนผู้บริจาค (เฉพาะบิลที่ตรวจสอบแล้ว)
   - นำสถิติไปแสดงผลแบบเรียลไทม์ที่หน้าแรก
3. **ระบบติดตามสถานะผู้บริจาค (Tracking Dashboard):**
   - สร้างหน้าเพจสาธารณะ `/tracking` สำหรับให้ผู้ใช้เช็คสถานะ
   - เพิ่มฟังก์ชันค้นหาประวัติการบริจาคด้วย "เบอร์โทรศัพท์"
   - สร้าง API `server/api/donations/track.ts` สำหรับรองรับการค้นหา
4. **ปรับปรุง UI แอดมิน (Admin UI):**
   - ย้ายกล่องสถานะเชื่อมต่อฐานข้อมูล (Database Connection) ไปไว้เป็นปุ่มลอย (Floating) ที่มุมขวาล่างของหน้าจอ
5. **ระบบรายงานสรุป (Admin Report):**
   - สร้างหน้า `/admin/report` สำหรับผู้ดูแลระบบ
   - เพิ่มระบบ Filter แยกดูข้อมูลตาม **"รายวัน"** และ **"รายเดือน"**
6. **รูปแบบการพิมพ์รายงาน (Print Report):**
   - จัดทำระบบหน้าจอสำหรับสั่งพิมพ์รายงานในรูปแบบกระดาษ A4 (ใช้ `@media print`) พร้อมสรุปยอดเงินรวม
7. **ระบบพิมพ์ใบเสร็จ (Print Receipt):**
   - ปรับปรุงหน้าต่าง Success Modal ในหน้า `/donate`
   - เพิ่มปุ่ม "พิมพ์ใบเสร็จ / บันทึกเป็น PDF" พร้อมจัดหน้ากระดาษให้สวยงาม
8. **ระบบพิมพ์ใบกำกับภาษี (Tax Invoice):**
   - เพิ่มระบบออกใบกำกับภาษีชั่วคราว โดยเพิ่มปุ่มพิมพ์แยกออกมาหากผู้ใช้กรอกข้อมูลภาษี
9. **หมายเหตุการจัดส่งเอกสาร:**
   - เพิ่มข้อความ *"หมายเหตุ: ใบกำกับภาษีฉบับจริงจะถูกจัดส่งทางไปรษณีย์ภายใน 7-14 วันทำการ"* ในขั้นตอนขอเอกสาร

**การแก้ไขบั๊กที่พบระหว่างทาง:**
- **Fixed:** แก้ไขบั๊ก `Cannot access 'renderer' before initialization` ในระบบ Nuxt/Nitro โดยปรับปรุงการ Import และเรียกใช้ `PrismaClient` ในไฟล์ API ใหม่ให้เรียกใช้งานจาก `utils/prisma` ส่วนกลาง เพื่อป้องกันการเรียกซ้ำซ้อนในโหมด Dev
- **Fixed:** แก้ไขบั๊ก Syntax Error (Invalid end tag) ในไฟล์ `AppFooter.vue` โดยเปลี่ยนแท็กปิด `</div>` ที่เกินมาให้เป็น `</footer>` ที่ถูกต้อง


### [อัปเดตครั้งที่ 1] - วันที่สร้างโปรเจกต์
**รายละเอียด:** พัฒนาโครงสร้างระบบพื้นฐาน
**รายการที่ดำเนินการ:**
- ติดตั้งและตั้งค่า Nuxt 4 + Vue 3
- เชื่อมต่อระบบฐานข้อมูล PostgreSQL ผ่าน Prisma ORM
- พัฒนาระบบรับบริจาคออนไลน์ พร้อมอัปโหลดสลิปผ่าน Supabase Storage
- สร้างหน้า Admin Dashboard พื้นฐานสำหรับการจัดการ
